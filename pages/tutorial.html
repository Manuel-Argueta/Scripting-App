Welcome to the Scripting with PEPE Tutorial.

In this level you will learn the basics of programming in PEPE. This fun and simple language will allow you to control the Pepe The Frog character through scripts that you write! Here are the basics:

  • Each command that you write MUST begin with the keyword "GO" (case-sensitive)
  • Each command must specify what direction the character will move: "U" to move UP, "D" to move DOWN, "R" to move RIGHT, and "L" to move LEFT.
  • Each command must use parenthesis DIRECTLY AFTER specifying the direction of movement. You can specify the # of REPETITIONS (as a NUMBER) inside of these parameters as an argument OR leave it BLANK and default the # of REPETITIONS to 1.
  • VERY IMPORTANT: Each command MUST be written on a new line in order for it be considered seperate to the previous command. There is no need for annoying SEMICOLONS like other languages (..psst Java) Failing to seperate these commands will result in a program that does not work as intended or may not work at all.
