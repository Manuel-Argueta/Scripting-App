Welcome to the Scripting with DOGE Tutorial.

You will learn how to program with DOGE. Every command moves the DOGE character.

  • Start every command (line) with the word "GO" (ALL CAPITAL LETTERS)
  • Every command must have a direction using ONE LETTER (ALL CAPITAL LETTERS)
    • U -> Moves DOGE UP
    • D -> Moves DOGE DOWN
    • L -> Moves DOGE LEFT
    • R -> Moves DOGE RIGHT
  • After putting the direction use paranthesis "()" with a number inside. The number is how many times you want DOGE to move in that direction.
  • VERY IMPORTANT: Each command MUST be written on a new line.
